## Los conceptos básicos de React incluyen:

Componentes: piezas de código reutilizables que se utilizan para construir la interfaz de usuario de una aplicación.

Programación declarativa: en lugar de decirle al computador cómo realizar una tarea, se le da a React un conjunto de reglas y datos y se le permite determinar cómo actualizar la interfaz de usuario en función de esos datos.

Vínculo de datos: permite especificar qué datos necesita un componente para funcionar y cómo debe reaccionar cuando esos datos cambien.

Virtual DOM: una representación en memoria de la estructura de la interfaz de usuario de una aplicación. React utiliza el Virtual DOM para determinar qué cambios hacer en la interfaz de usuario cuando los datos cambian.

JSX: una extensión de JavaScript que permite escribir código que se asemeja al HTML y se integra con React para facilitar la creación de componentes.
